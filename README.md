# PS3-WOL-GUI
A GUI to manage your PS3.

### Overview
PS3-WOL-GUI is a utility to manage your PS3. It can connect to a PS3, turn it on or off, see the current power state
of the console, manage the thermals and the fans. It has the ability to set custom thresholds for the Dynamic fan setting,
and custom fan speeds for the Manual mode.

### Downloads
PS3-WOL-GUI is only available for Windows. Linux and Mac builds are not planned for the moment.
and can be downloaded from the [releases page](https://github.com/dorian-osly/PS3-WOL-GUI/releases).
